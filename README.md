Interactive analytics platform focused on applied data analysis in functional nutrition and chronobiology, combining structured datasets with exploratory visualization.

## Tech Stack
- Python 3.10+
- Streamlit
- Pandas
- Plotly Express
- st-aggrid

## How to run
```bash
git clone https://github.com/GabriGuerra/nutri-analytics.git
cd nutri-analytics
pip install -r requirements.txt
streamlit run src/app.py
```
