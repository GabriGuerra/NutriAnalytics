# NutriAnalytics

Interactive analytics platform for applied data analysis in functional nutrition and chronobiology.

## Tech Stack
- Python 3.10+
- Streamlit
- Pandas
- Plotly Express
- st-aggrid

## How to run
```bash
git clone https://github.com/GabriGuerra/nutri-analytics.git
cd nutri-analytics
pip install -r requirements.txt
streamlit run src/app.py
```
